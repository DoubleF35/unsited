Unsited

Connect your Google Maps key

Unsited runs entirely in your browser and talks to Google directly. Paste an API key with Maps JavaScript API and Places API (New) enabled. The key is kept in this browser only — it is never sent anywhere else or stored in the project.

Restrict the key to your site's URL before going live. Setup steps →

How to get a key (2 minutes)
  1. Open Google Cloud Console and pick or create a project (a billing account must be attached).
  2. In APIs & Services → Library, enable Maps JavaScript API and Places API (New). Both are required.
  3. In Credentials → Create credentials → API key, copy the key.
  4. Edit the key → Application restrictions → Websites → add your address, e.g. https://yourname.github.io/*.
  5. Under API restrictions, limit it to the two APIs above. Save, then paste it here.

Heads up on cost: detecting "no website" forces Google's Enterprise tier (~$35 per 1,000 calls). You get 1,000 free calls a month. Each area sweep is one call per grid cell — Unsited shows the count before you run it.
